ElementUtils

ElementUtils

ElementUtils creates DOM wrapper for JSactions core Objects such as View, ViewNavigator, ViewStack, Component, Container

JSactions core object -> HTML DOM wrapper

ViewNavigator HTML DOM wrapper: < div class="jsact-viewnavigator ${navigatorId}" >

ViewStack HTML DOM wrapper: < div class="jsact-viewstack ${viewstackId}" >

View HTML DOM wrapper: < div class="jsact-view ${viewId}" >

Container HTML DOM wrapper: < div class="jsact-container ${containerId}" >

Component HTML DOM wrapper: < div class="jsact-component ${componentId}" >
